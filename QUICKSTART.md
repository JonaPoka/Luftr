## Step 1: Open in Android Studio

1. Download and install [Android Studio](https://developer.android.com/studio)
2. Clone or download this repository
3. Open Android Studio
4. Click **"Open"** and select the `Luftr` folder
5. Wait for Gradle sync to complete (may take a few minutes on first run)

## Step 2: Run the App

### Option A: Using an Emulator
1. In Android Studio, go to **Tools > Device Manager**
2. Click **"Create Device"**
3. Select a phone (e.g., "Pixel 6")
4. Download a system image (API 34 recommended)
5. Finish creating the emulator
6. Click the **â–¶ Run** button
7. Select your emulator
8. Wait for the app to install and launch

### Option B: Using Your Phone
1. Enable Developer Options on your Android device:
   - Go to **Settings > About Phone**
   - Tap **Build Number** 7 times
2. Enable USB Debugging:
   - Go to **Settings > Developer Options**
   - Enable **USB Debugging**
3. Connect your phone via USB
4. Click the **â–¶ Run** button in Android Studio
5. Select your device
6. Wait for the app to install and launch

## Step 3: Your First Workout

### Create a Custom Workout
1. On the home screen, tap the **+** button
2. Enter a name (e.g., "Chest Day")
3. Tap **"Start Workout"**
4. Tap the **+** button to add an exercise
5. Enter exercise name (e.g., "Bench Press")
6. Select muscle group (e.g., "Chest")
7. Tap **"Add Exercise"**
8. Tap **"Add Set"** on the exercise card
9. Enter reps (e.g., 10) and weight (e.g., 60)
10. Tap **"Add Set"** to save
11. Add more sets or exercises as needed

## Troubleshooting

### Gradle Sync Failed
- Check internet connection
- In Android Studio: **File > Invalidate Caches > Invalidate and Restart**

### App Won't Install
- Make sure USB debugging is enabled
- Try unplugging and replugging your device
- Restart Android Studio

### App Crashes
- Check minimum Android version (8.0 required)
- Update your device if needed
- Check Logcat in Android Studio for errors
